# Sensor Options for Excavation Terrain Monitoring

## Purpose

This report compares stereo, ToF, and LiDAR sensors for a near-real-time terrain-monitoring setup around excavation equipment. The main evaluation criteria are ruggedization, usable distance, spatial resolution, and practical value for mapping uneven soil, trench geometry, spoil piles, and changing excavation surfaces.
